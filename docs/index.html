Features

Everything you need

From magnet URL to completed download — fully automated, with full control.

🔄
Automatischer Lifecycle
Upload → Poll → Unlock → aria2 → Done → Discord. Every step is monitored, errors are detected and resolved automatically.
🎯
Slot-basierte Queue
Konfigurierbares Concurrent-Download-Limit. aria2-Requests werden serialisiert mit 50ms Mindestabstand — kein Request-Burst.
🗄️
SQLite & PostgreSQL
SQLite out-of-the-box, PostgreSQL for multi-container setups. Automatic fallback and bidirectional migration.
🔔
Discord Rich Embeds
Bot name, avatar and separate webhook for "Added" events. Anti-spam deduplication and rate-limiting.
🤖
FlexGet Integration
FlexGet v3 REST API. Trigger tasks manually or on a schedule with configurable jitter. Webhook events for every phase.
📊
Statistik & Reporting
Comprehensive metrics with configurable time window (1h to ~1 year), JSON export and periodic snapshots.
🔁
Full-Sync
Regular reconciliation of all torrents against AllDebrid — detects ready torrents stuck locally as error/queued.
🚫
File Filters
Block by extension, keyword and minimum size. Filtered files are reported via Discord.
📺
Sonarr & Radarr
Import trigger after completed download. Automatic library update.

Installation

Up and running in 2 minutes

Docker Compose is the recommended way. SQLite is preconfigured — no database setup needed.

🐳 Docker Compose
git clone https://github.com/kroeberd/\
  alldebrid-client.git
cd alldebrid-client
docker compose up -d

# → http://localhost:8080
🐋 Docker run
docker run -d \
  --name alldebrid-client \
  --restart unless-stopped \
  -p 8080:8080 \
  -v /config:/app/config \
  -v /downloads:/download \
  kroeberd/alldebrid-client:latest
🗄️ With PostgreSQL
# docker-compose.yml
environment:
  DB_TYPE: postgres

# Dann in Settings → Database
# Configure connection
⚙ Environment variables
CONFIG_PATH=/app/config/config.json
DB_PATH=/app/data/alldebrid.db
TZ=Europe/Berlin
DB_TYPE=postgres     # optional
LOG_LEVEL=DEBUG      # optional
